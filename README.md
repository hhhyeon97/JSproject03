### user story

1. 사용자는 뉴스 목록을 볼 수 있다.
2. 사용자는 네비게이션바를 통해 카테고리별 뉴스를 확인할 수 있다.
3. 사용자는 뉴스 이미지, 제목, 내용 중 하나를 클릭하여 해당 뉴스 링크로 이동할 수 있다.
4. 사용자는 원하는 키워드로 검색 기능을 사용할 수 있다.
5. 사용자는 하단의 페이지바를 통해 페이지 이동이 가능하다.
